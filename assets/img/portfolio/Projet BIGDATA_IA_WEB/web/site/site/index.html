Le projet consiste à concevoir et développer une application web pour la gestion du patrimoine arboré, en tant que preuve de concept pour la ville de Saint-Quentin (Aisne). Le but est d'approfondir les compétences des étudiants en Big Data, Intelligence Artificielle (IA), Développement Web, et Bases de Données à travers une application complète de traitement et de visualisation de données.
